# ATtiny85

Projets basés sur la platine d'essais à laquelle peuvent être ajoutés divers périphériques connectés sur un [bus I²C](https://fr.wikipedia.org/wiki/I2C):
- Capteur de température (cf. ![DS18B20](DS18B20))
- Capteur d'humidité
- Horloge RTC
- Mémoire EEPROM
- etc.
