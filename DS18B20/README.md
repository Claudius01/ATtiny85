# DS18B20

Projet basé sur la platine d'essais permettant de gérer 5 capteurs de température [DS18B20](https://www.analog.com/media/en/technical-documentation/data-sheets/ds18b20.pdf) avec un ATtiny85 cadencé à 20 MHz.

![Platine d'essais DS18B20](Platine-ATtiny85-DS18B20.png)

La gestion est faite au dessus du micro-OS
