# Liste des commandes / réponses supportées par uOS
# -------------------------------------------------

@ => les versions de uOS et/ou de l'EEPROM peuvent changer suivant l'état de développement
@ => Les Ids de réponses (ie. # 4080 peuvent varier suivant les copier/coller des commandes/réponses)
@ => Les présentations comme:
@           Addr     0    1    2    3     4    5    6    7    0 1 2 3 4 5 6 7
@    - "28: 0030  4154 7469 6e79 3835  5f75 4f53 2024 5265  |ATtiny85_uOS $Re|"
@
@           Addr     0    1    2    3     4    5    6    7    0 1 2 3 4 5 6 7
@    - "80: 03ea  b0d2 0895 2323 2320  4545 5052 4f4d 3a20  |....### EEPROM: |
@
@    sont effectuées par le terminal qui interprète les réponses de l'uOS

@ Emission du prompt au RESET
@ ---------------------------
# 4064: [[### ATtiny85_uOS $Revision: 1.1 $]]
# 4065: [[### EEPROM: 1.2.6]]
# 4066: [[### Type: 78]]
# 4067: [[### Id: 00]]

@ Réponses à la commande "<f"
@ ---------------------------
@ Lecture de la signature et des fusibles
@ ---------------------------
>>> Send command [<f] (2 bytes)
# 4080: [[02>f0000]]
# 4082: [[0x1e][0x93][0x0b]]
# 4083: [[0xf1][0xff][0xff][0xd7]]

@ => Signature: [0x1e09300b] (ATtiny85)
@ => Fusibles:
@    => Low:      [0xf1]
@    => Lock:     [0xff]
@    => Extended: [0xff]
@    => High:     [0xd7]
@ ---------------------------
